Для того чтобы заргестрироваться нужно перейти по следующему адресу и заполнить необходимые поля:
http://127.0.0.1:8000/auth/users/
После этого необходимо сгенерировать jwt токен c помощью username и password с предыдущего шага:
http://127.0.0.1:8000/auth/jwt/create/

Сохранить себе access и refresh токен для дальнейшего использования

В случае если access просрочился отправить запрос на его обновление с помощью refresh токена по следующему адресу
http://127.0.0.1:8000/auth/jwt/refresh/
