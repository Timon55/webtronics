для того чтобы лайкнуть пост нужно отправить post запрос с jwt токеном для авторизации по следующему адресу:
http://127.0.0.1:8000/api/posts/{pk}/like/ - где pk - это id поста который хочется лайкнуть
http://127.0.0.1:8000/api/posts/{pk}/unlike/ - аналогично только если нужно забрать лайк
http://127.0.0.1:8000/api/posts/{pk}/fans/ - чтобы получить список пользователей лайкнувших данный пост
